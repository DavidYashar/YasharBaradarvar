to connect the react to php server:
php -S localhost:8000

to use jquery in react:
npm install jquery


to change the react level to production:
npm run build

to check if the build app works:
npm install -g serve
serve -s build

to download and install router:
npm install --save react-router-dom


to install axios
npm install axios
